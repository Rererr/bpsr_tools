# スターレゾナンス パワーコア計算機

ブルーアーカイブ スターレゾナンス（BPSR）のパワーコア効果を計算するツールです。

## 機能

- 4つのパワーコアスロットに効果を設定
- 各スロットに3つの効果を追加可能
- 効果の合計値を自動計算
- 内訳の表示

## 使い方

1. パワーコアタイプを選択（攻撃型/支援型/防御型）
2. パワーコアを選択
3. 効果タイプと数値を入力
4. 計算結果を確認

## 開発

```bash
# 依存関係のインストール
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build
```
